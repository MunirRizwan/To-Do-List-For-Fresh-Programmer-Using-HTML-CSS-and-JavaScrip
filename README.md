# To-Do-List-For-Fresh-Programmer-Using-HTML-CSS-and-JavaScrip

This is a To-Do List small Assignment for fresh developers using HTML5, CSS, and JavaScript.

In this to-do list, I built three main logic/operations
1) Add Task at start position or end position
2) Delete Task from start or end
3) Sort tasks in Ascending order or descending order.

for simplicity/easiness, I wrote all HTML code, CSS code, JavaScript code in one file (ToDo List.html)
